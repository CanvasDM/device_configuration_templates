# Canvas Device Manager BT610 Configuration Templates

This repository contains example configurations for the Laird Connectivity Canvas enabled [BT610][BT610] IoT sensor device. It is intended to provide guidance on device configuration, and as a starting point to development of bespoke user configuration files.

The following examples are available.

[Current - Four 4-20mA current sensors][Current_-_Four_4-20mA_current_sensors]

[Current - Three 500A current sensors on AIN1 through AIN3][Current_-_Three_500A_current_sensors_on_AIN1_through_AIN3]

[Pressure - Single pressure sensor on AIN1][Pressure_-_Single_pressure_sensor_on_AIN1]

[Pressure and ultrasonic - Pressure sensor on AIN1 and ultrasonic sensor on AIN4][Pressure_and_ultrasonic_-_Pressure_sensor_on_AIN1_and_ultrasonic_sensor_on_AIN4]

[Thermistor - Four thermistors on TH1 through TH4][Thermistor_-_Four_thermistors_on_TH1_through_TH4]
